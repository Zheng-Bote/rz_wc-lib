rz-header

Displays a fixed header with a given brand logo and a given brand name.


Syntax

Within HTML header: 
<script type="module" src="/assets/rz-header.js"></script>
Within HTML body: 
<rz-header> brand_img="path/to/image" brand_title="my title" <rz-header/>

default

<rz-header>
  brand_img="img/icons/home-outline.svg"
  brand_title="Home"
<rz-header/>

Attributes

attribute value description
brand_img <path/to/image> e.g.: brand_img="img/icons/home-outline.svg"
brand_title <brand title> eg: brand_title="Home"
