rz-online

Displays the online status in red (offline) and green (online)

Usefull within a PWA


Syntax

Within HTML header: 
<script type="module" src="/assets/rz-online.js"></script>
Within HTML body: 
<rz-online attribute="value" > <rz-online/>

default

<rz-online>
  with_text=""
<rz-online/>

Attributes

attribute value description
with_text <yes>|<no> display text: online|offline
