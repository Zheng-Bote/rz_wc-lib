Menu

web-components


rz-navmenu


Syntax

Within HTML header: 
<script type="module" src="/assets/rz-navmenu.js"></script>
Within HTML body: 
<rz-navmenu attribute="value" > <rz-navmenu/>

default

<rz-navmenu>
  json_url="/data/rz-navmenu.json"
  current_id="navmenu"
<rz-navmenu/>

Attributes

attribute value description
json_url <URL> URL to JSON file, eg: "/data/rz-navmenu.json"
current_id <id> id of the current page, e.g.: "navmenu"
