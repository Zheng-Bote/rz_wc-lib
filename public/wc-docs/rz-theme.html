rz-theme

Switch between dark mode and light mode.

In fact, the web-component just sets the following attributes:

  • document element attribute "data-theme"
  • document element style "color-scheme"
  • local storage "theme"

Syntax

Within HTML header: 
<script type="module" src="/assets/rz-theme.js"></script>
Within HTML body: 
<rz-theme> <rz-theme/>

default

<rz-theme>
<rz-theme/>

attribute value description

Usage

Example usage

:root {
  color-scheme: light dark;
  --font-color: light-dark(oklch(0% 0 0), oklch(95.84% 0.004 214.33));
  --background-color: light-dark(#cae5ff, #6f8ab7);
}
body {
  background-color: var(--background-color);
  color: var(--font-color);
}
