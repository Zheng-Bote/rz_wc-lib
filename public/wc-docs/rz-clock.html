rz-clock

Displays an analog clock with the given timezone and locale.


Syntax

Within HTML header: 
<script type="module" src="/assets/rz-clock.js"></script>
Within HTML body: 
<rz-clock attribute="value" > <rz-clock/>

default

<rz-clock
  clock_width="300"
  timezone="Asia/Shanghai"
  title="Beijing"
>
<rz-clock/>

Attributes

attribute value description
width <pixel> eg: "150"
timezone <timezone> timezone, eg: "Asia/Shanghai"
title <name> "Beijing"
