rz-datetime

Displays the date / time based on given timezone and locale.


Syntax

Within HTML header: 
<script type="module" src="/assets/rz-datetime.js"></script>
Within HTML body: 
<rz-datetime> attribute="value"> <rz-datetime/>

default

<rz-datetime
  tz="Europe/Berlin"
  format="de-DE"
  fulldatetime="yes"
  show_tz_pre="no";
  show_tz_post="no";
  show_tz_top="no"
>
<rz-datetime/>

Attributes

attribute value description
tz <IANA time zone identifier> timezone, eg: "Europe/Berlin"
format <locales> locale, eg: "de-DE"
fulldatetime <yes> | <no> yes, eg: "12.4.2025, 19:28:46"
justdate <yes> | <no> yes, eg: "12.4.2025"
justtime <yes> | <no> yes, eg: "19:28:46"
show_tz_pre <yes> | <no> show timezone before date/time
show_tz_post <yes> | <no> show timezone after date/time
show_tz_top <yes> | <no> show timezone on-top of date/time
