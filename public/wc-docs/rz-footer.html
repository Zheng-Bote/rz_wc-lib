rz-footer

Displays a fixed footer with three sections (left, middle, right).
If the given year is smaller than the current year, string will be "<given year>-<current year>"



Syntax

Within HTML header: 
<script type="module" src="/assets/rz-footer.js"></script>
Within HTML body: 
<rz-footer> attribute="value" <rz-footer/>

default

<rz-footer>
  url_left="/contact.html"
  url_left_title="Contact";
  url_middle="/about.html";
  url_middle_title="about";
  url_right="/legal.html";
  url_right_title="Legal Notice";
  name="ZHENG Robert";
  year="2023";
  version="0.2.0";
<rz-footer/>

Attributes

attribute value description
url_left <URL> e.g.: url_left="/contact.html"
url_left_title <title> eg: url_left_title="Contact"
url_middle <URL> e.g.: url_middle="/about.html"
url_middle_title <title> eg: url_middle_title="About"
url_right <URL> e.g.: url_right="/legal.html"
url_right_title <title> eg: url_right_title="Legal Notice"
name <copyright holder> eg: name="Bugs Bunny Productions"
year <YYYY> eg: year="2023"
