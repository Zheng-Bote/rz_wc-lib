Syntax

Within HTML header: 
<script type="module" src="/assets/rz-button-top.js"></script>
Within HTML body: 
<rz-datetime> <rz-datetime/>

default

<rz-datetime>
  tz="Europe/Berlin"
  format="de-DE"
  fulldatetime="yes"
<rz-datetime/>

Attributes

attribute value description
tz <timezone> timezone, eg: "Europe/Berlin"
format <locales> locale, eg: "de-DE"
fulldatetime <yes> | <no> yes, eg: "12.4.2025, 19:28:46"
justdate <yes> | <no> yes, eg: "12.4.2025"
justtime <yes> | <no> yes, eg: "19:28:46"
